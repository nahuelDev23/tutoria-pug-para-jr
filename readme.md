## Glosario

- mb: margin-bottom

### padding vs margin

El padding es interno
El margin es la distancia entre elementos hermanos

## Flex

Si quiero poner dos elementos hermanos uno arriba del otro
y que tenga una separacion lo hago asi: `.flex.flex-col.gap-4`
